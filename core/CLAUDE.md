# CLAUDE.md

Project-level instructions for Claude Code sessions.

## Project Overview

This repository contains a collection of agentic AI tools organized into skills, hooks, agents, prompts, MCP servers, workflows, and core configuration files.

## Conventions

- Each tool lives in its own named subdirectory under the appropriate category
- Every tool directory includes a `README.md` describing its purpose and usage
- Prompt templates use `{{parameter}}` syntax for placeholders
- Hook scripts should be POSIX-compatible shell
